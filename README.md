# Dotfiles

## Installation

install gnu stow then

```sh
stow */
```

## Some stuff I like to have

__IMPORTANT:__ `sudo apt install build-essential` & `sudo apt install stow`

1. [GitHub CLI](https://github.com/cli/cli) Makes life easier
2. [Rust + Cargo with rustup](https://www.rust-lang.org/tools/install) Needed for bob
3. [bob](https://github.com/MordechaiHadad/bob) Makes NeoVim more swag
4. [NodeJS via nvm](https://github.com/nvm-sh/nvm) Needed for a bunch of NeoVim stuff
5. RipGrep: `sudo apt install ripgrep` A lot of my scripts use this
6. FZF: `sudo apt install fzf` Same reason as above
