# Dotfiles

## Installation

Install `gnu stow` then

```sh
stow */
```

## Some stuff I like to have

**IMPORTANT:** `sudo apt install build-essential` & `sudo apt install stow`

1. [GitHub CLI](https://github.com/cli/cli) Makes life easier (I only use this for initial auth on a new system)
2. [Rust + Cargo with rustup](https://www.rust-lang.org/tools/install) Needed for bob and crabs are cute :3
3. [bob](https://github.com/MordechaiHadad/bob) Makes NeoVim more swag
4. [NodeJS via nvm](https://github.com/nvm-sh/nvm) Needed for a bunch of NeoVim stuff, also JavaScript is cool I guess
5. [ripgrep](https://github.com/BurntSushi/ripgrep): RIP grep, I guess
6. [fzf](https://github.com/junegunn/fzf?tab=readme-ov-file#using-git): Can't live without it
7. [zoxide](https://github.com/ajeetdsouza/zoxide): Better `cd` for a civilized era
8. [lazygit](https://github.com/jesseduffield/lazygit?tab=readme-ov-file#ubuntu): Best Git CLI
9. [ZelliJ](https://zellij.dev/): Sorry tmux, trying something new for now
10. [LazyVim](https://www.lazyvim.org/): Sane defaults and I'm lazy, so it fits
